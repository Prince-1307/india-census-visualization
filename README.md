# 📊 India Census 2011 - Data Visualization Dashboard

An interactive data visualization dashboard built using **Python, Plotly, and Streamlit** to explore insights from the **India Census 2011 dataset**.

This project focuses on making complex census data **interactive, intuitive, and easy to analyze** through dynamic visualizations.

---

## 🌐 Live Application

---

## 🚀 Features

- Select any **State** or view **Overall India**
- Compare two parameters dynamically
- Interactive **map-based visualization**
- Hover-based detailed insights
- Clean and user-friendly interface

---

## 📊 Parameters Available

The dashboard allows comparison between multiple census metrics:

- Population  
- Male / Female Population  
- Literate / Male Literate / Female Literate  
- Literacy / Male Literacy / Female Literacy  
- Sex Ratio  
- SC / ST Population  
- Employed / Unemployed  
- Households with Electricity  
- Households with Internet  
- Households with Computer
